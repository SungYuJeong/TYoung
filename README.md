"Trendy Word"
=========
- ### Main Function

  - 아리랑 홈페이지("http://arirang.com") 뉴스들을 주제별로 크롤링한 후 
    단어들을 수집하고 가공하여 주요 이슈별 트렌디한 영어단어들을 제공

  - 단어들의 카운트롤 누적해 랭킹별 영어 단어들을 목록화하여 보여줌
  
  - 각 단어들의 발음, 의미, 예문 등을 제공하고 사용자가 발음한 내용을 체크할 수 있음

  - 워드클라우드를 활용해 단어들을 시각화하고 트렌디한 단어 변화를 볼 수 있음
